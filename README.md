# Earthquake Damage Prediction

## Overview

This project predicts earthquake damage severity using machine learning models and structural data analysis.

## Features

* Data preprocessing
* Feature analysis
* Model training and evaluation
* Damage prediction
* Performance visualization

## Tech Stack

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

## Workflow

1. Load earthquake dataset
2. Perform preprocessing
3. Train prediction model
4. Evaluate model performance
5. Predict structural damage levels

## Project Structure

* `earthquake_damage_prediction.ipynb`
* `requirements.txt`
* `README.md`
